# AC學期2-2_A12_Q1_MovieList_getGrid&List

AlphaCamp學期2-2，A12作業專案的MovieList_getGrid&List，  
使用 VS code 打造的電影清單。

## 功能描述

- 搜尋電影標題的功能。
- 可加入電影清單中的電影，並在Favorite頁面中看到，也可以刪除已加入的電影。
- 可以切換「清單模式」和「卡片模式」
